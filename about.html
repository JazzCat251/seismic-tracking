About Seismic Tracker

A modern tool to visualize, explore & stay informed about earthquakes across the globe.

Seismic Tracker is a web application designed to monitor global earthquake activity in real time. It integrates live data from the USGS Earthquake API to display seismic events on an interactive map.

Purpose

The purpose of this project is to provide a visual and educational tool to explore earthquake trends, understand their distribution, and raise awareness of seismic risks across regions.
